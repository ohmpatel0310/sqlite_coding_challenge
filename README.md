# Business Data SQL Suite

### 📊 Project Overview
This project contains a comprehensive suite of SQL queries designed to extract actionable insights from a retail database. I developed these queries to solve specific business problems, such as identifying high-value customers, analyzing product distribution, and ranking regional performance.

### 🛠️ Technologies Used
* **Database Engine:** SQLite
* **Language:** SQL
* **Tools:** SQLite3, GitHub

### 🚀 How to Run the Queries
1. Clone the repository: 
   `git clone https://github.com/ohmpatel0310/business-data-SQL-suite.git`
2. Open the `bais_sqlite_lab.db` file using any SQLite-compatible database manager.
3. Run the queries found in `challenge.sql` to view the analysis results.

### 💡 Key Technical Insights
Throughout this project, I implemented several advanced SQL techniques:
* **Multi-Table Joins:** Combined customer, order, and product tables to view the full lifecycle of a transaction.
* **Subqueries & Aggregations:** Used nested logic to calculate loyalty-level distributions and filter for top-performing cities.
* **Case Logic:** Applied conditional logic to categorize customers based on spending habits and loyalty rankings.
